Jekyll 搭建静态博客

前言

我的博客是用jekyll搭建的，这个由于我最先接触到别人的也是用jekyll来搭建的，先入为主，我也跟着用jekyll来搭建了，其实网络上还有一种是用Hexo来搭建的，我没试过这个工具，在此我就不写关于Hexo的教程了，以下为jekyll搭建静态博客教程。

正文

在jekyll的官网上 http://jekyllrb.com/ 其实已经说得比较明白了，我在这里还是简单的说一下吧。我用的是Windows系统。
主要环节有：安装Ruby，安装RubyGems，安装jekyll，安装代码高亮插件，安装node.js

安装Ruby

ruby官网下载安装：https://www.ruby-lang.org/en/downloads/

注意：高版本可能有好多问题，建议安装如下版本并安装相应的Devkit

ruby 2.2.6p396 (2016-11-15 revision 56800) [x64-mingw32]

在cmd或者powershell中，输入一下命令

ruby -v

若能够得到ruby版本号，即安装成功
如果不行则在系统变量中自行添加ruby的bin路径即可

安装RubyGems

官网下载 http://rubygems.org/pages/download rubygems.zip
解压后cd到RubyGems目录
或者在RubyGems目录下按住shift键+鼠标右键即可在菜单中出现powershell选项
执行安装

用RubyGems安装Jekyll

执行下面的语句安装

gem install jekyll

至此jekyll就已经安装完毕了，后续就是个性化的自己设定了。

创建博客

在d盘新建一个工作区jekyllWorkspace

cd到jekyllWorkspace

执行jekyll new name创建新的工作区

cd到博客文件夹，开启服务器

jekyll serve

如果命令运行出错，改用 bundle exec jekyll serve

tips：

  • 可以直接到jekyllthemes的网站上下载自己喜欢的博客模板

后记

  • 整个安装过程参考了jekyll官网，注意jekyll还有一个简体中文官网，有些内容没有翻译过来，有可能会走弯路，建议如果想看中文的相关资料，也要中英对照着阅读。jekyll中文网 http://jekyllcn.com, jekyll英文网 http://jekyllrb.com
  • jekyll中的css是用sass写的，当然直接在_sass/_layout.scss中添加css也是可以的。
  • 本文是用Markdown格式来写的，相关语法可参考： Markdown 语法说明 (简体中文版) http://wowubuntu.com/markdown/
  • 按照本文的说明搭建完博客后，用github Pages托管就可以看到了。
  • 博客默认是没有评论系统的，详细安装办法可访问多说官网 http://duoshuo.com/，当然也可以使用搜狐畅言 http://changyan.sohu.com/作为评论系统。
  • 博客网站可以绑定自己的域名，如果没有域名，可到阿里云等提供域名购买的地方自行购买。
